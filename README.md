# PromisesPlugin

ES6-Promises polyfill for Cordova/PhoneGap

# Configure your plugin

Add as a dependency in the plugin.xml:

```
<dependency
  id="com.vladstirbu.cordova.promise"
  url="https://github.com/vstirbu/PromisesPlugin.git">
</dependency>
```
